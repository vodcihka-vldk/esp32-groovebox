## 📐 Схема и макет PCB

### 🎯 PCB Design Specifications

**Параметры PCB:**
- **Размер:** 100mm × 80mm × 1.6mm
- **Слои:** 2-layer (Top/Bottom)
- **Материал:** FR-4, TG 130°C
- **Толщина меди:** 1oz (35µm)
- **Минимальный трек:** 0.2mm
- **Минимальное отверстие:** 0.3mm
- **Паяльная маска:** Green
- **Маркировка:** White silkscreen

### 📍 Компоновка компонентов

```
┌─────────────────────────────────────────────────────────┐
│  [LCD Display]        [Encoder]                    │
│  16x2                 (Rotary)                     │
│                                                      │
│  [Button Matrix]      [Status LEDs]                 │
│  4x4                  (8x indicator)              │
│                                                      │
│  [ESP32]             [Audio Section]                │
│  (Main MCU)           (DAC + Op-amps)              │
│                                                      │
│  [Power Supply]       [Connectors]                  │
│  (5V/3.3V LDO)        (Audio/MIDI/SD)          │
│                                                      │
│  [SD Card Slot]      [Optional OLED]               │
│  (SPI)                (I2C)                         │
└─────────────────────────────────────────────────────────┘
```

### 🔌 Схема питания

```
5V DC Input
    │
    ├─► [LDO 3.3V] ──► ESP32 Core
    │                    ├─► LCD Display
    │                    ├─► SD Card
    │                    └─► Digital Logic
    │
    ├─► [PCM5102 DAC] ──► Audio Output
    │
    └─► [NE5532 Op-amps] ──► Audio Buffer
```

### 🎵 Аудио тракт

```
ESP32 I2S ──► PCM5102 DAC ──► NE5532 Buffer ──► 3.5mm Jack
     │              │                │              │
  24-bit/192kHz   Stereo          Low Noise     Line Level
```

### 🔄 Real-time Processing Flow

```
Audio Interrupt (1kHz)
┌─────────────────────────────────────────────────────────┐
│ 1. Read Sequencer Position                           │
│    └─► Determine active steps                       │
│                                                    │
│ 2. Trigger Active Voices                             │
│    └─► Start/stop sample playback                   │
│                                                    │
│ 3. Process Active Voices                            │
│    └─► Read sample data from PSRAM                  │
│    └─► Apply pitch/volume/panning                 │
│                                                    │
│ 4. Mix and Apply Effects                          │
│    └─► Sum all voices                            │
│    └─► Apply DSP effects                         │
│                                                    │
│ 5. Output to I2S                                  │
│    └─► Send to DAC                              │
└─────────────────────────────────────────────────────────┘
```

## 📖 Описание проекта (маркетинг)

### 🎯 **ESP32 Groovebox - Профессиональный драм-синтезатор с открытым кодом**

#### 🚀 **Уникальное торговое предложение**

**"Профессиональные возможности по цене DIY проекта"**

ESP32 Groovebox - это революционное устройство, которое сочетает в себе:
- **Мощность профессиональных groovebox** (Roland TR-8S, Elektron Digitakt)
- **Доступность DIY проектов** (открытый код, кастомизация)
- **Современные технологии** (WiFi, Bluetooth, мобильное управление)

#### 🎵 **Ключевые преимущества**

**🏆 Профессиональное звучание:**
- 24-bit/192kHz аудио с PCM5102 DAC
- 8-голосная полифония с нулевой задержкой
- Продвинутые DSP эффекты (drive, reverb, delay, EQ)
- Multi-layer sampling с velocity-sensitive сэмплами

**🔧 Расширенные возможности:**
- Parameter locks для пер-шаговой автоматизации
- Real-time MIDI запись с квантизацией
- Performance modes (roll, flam, reverse, stutter)
- SD card поддержка с проектами (.gbx формат)

**📡 Современная conectivity:**
- WiFi с Ableton Link синхронизацией
- Bluetooth для мобильного управления
- USB MIDI Class для DAW интеграции
- Облачная синхронизация проектов

**🎛️ Интуитивный интерфейс:**
- Профессиональное меню (10 разделов)
- Rotary encoder с точной навигацией
- LCD/OLED дисплей с четкой индикацией
- Button matrix для быстрого доступа
