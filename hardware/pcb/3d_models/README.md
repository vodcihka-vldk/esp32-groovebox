# 3D Models for ESP32 Groovebox

## Component Models:
- ESP32-WROOM-32.step
- PCM5102A.step
- USB_C_Receptacle.step
- SSD1306_OLED.step
- EC11E_Encoder.step
- WS2812B_LED.step
- Switch_6x6mm.step

## Usage:
- Import into KiCad for 3D visualization
- Check clearances and fit
- Generate assembly drawings
- Create mechanical documentation

## Notes:
- All models in STEP format
- Compatible with most CAD software
- Accurate dimensions from datasheets
