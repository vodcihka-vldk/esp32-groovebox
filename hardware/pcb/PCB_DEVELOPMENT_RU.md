# 🔧 Разработка PCB для ESP32 Groovebox

## 📋 Обзор
Полное руководство по разработке печатной платы для ESP32 Groovebox.

## 🎯 Текущий статус

### ✅ Завершено:
- **Основные компоненты размещены**
- **ESP32-WROOM-32** - центральный процессор
- **PCM5102A** - I2S аудио DAC
- **USB-C разъем** - питание и связь
- **Аудио разъем 3.5мм** - выход на наушники
- **AMS1117-3.3** - стабилизатор питания
- **Монтажные отверстия** - M3 винты
- **Тестовые точки** - для отладки

### 🔄 Следующие шаги:

#### **1. Разводка дорожек (Traces)**
```
🔌 Питание (Power Net Class):
- +5V от USB-C к AMS1117-3.3
- +3.3V от AMS1117-3.3 ко всем компонентам
- GND - общая земля
- Ширина дорожек: 0.5mm
- Via: 1.0mm / 0.5mm

🎵 Аудио (Audio Net Class):
- I2S сигналы от ESP32 к PCM5102A
- Аудио выходы от PCM5102A к разъему
- Ширина дорожек: 0.15mm
- Via: 0.6mm / 0.3mm

📡 Цифровые сигналы (Default Net Class):
- I2C (SDA/SCL) к дисплею
- SPI к Flash памяти
- UART к программатору
- GPIO к кнопкам и энкодеру
- Ширина дорожек: 0.25mm
- Via: 0.8mm / 0.4mm

🔌 USB (USB Net Class):
- USB_D+ и USB_D- дифференциальная пара
- Ширина дорожек: 0.3mm
- Via: 0.8mm / 0.4mm
```

#### **2. Разводка по слоям**
```
🔵 Top Layer (F.Cu):
- Все компоненты
- Сигнальные дорожки
- Короткие соединения

🟢 Bottom Layer (B.Cu):
- Питание (+5V, +3.3V)
- Земля (GND)
- Длинные соединения
- Плоскости питания

📝 Слои маркировки:
- F.SilkS: названия компонентов
- B.SilkS: информация о плате
- F.Mask/B.Mask: паяльные маски
```

#### **3. Правила проектирования (DRC)**
```
📏 Минимальные размеры:
- Ширина дорожки: 0.15mm (аудио), 0.25mm (сигналы), 0.5mm (питание)
- Зазор между дорожками: 0.1mm (аудио), 0.2mm (сигналы), 0.15mm (питание)
- Размер via: 0.6mm (аудио), 0.8mm (сигналы), 1.0mm (питание)
- Отступ от края платы: 0.2mm

🔍 Проверка DRC:
- Короткие замыкания
- Обрывы дорожек
- Неправильные зазоры
- Отсутствующие соединения
```

## 🛠️ Пошаговая инструкция

### **Шаг 1: Подготовка**
1. ✅ Открыть `esp32_groovebox_complete.kicad_pro`
2. ✅ Проверить размещение компонентов
3. ✅ Убедиться в правильности footprint'ов

### **Шаг 2: Разводка питания**
1. 🔌 Развести +5V от USB-C к AMS1117-3.3
2. 🔌 Развести +3.3V от AMS1117-3.3 ко всем компонентам
3. 🔌 Создать плоскость GND на нижнем слое
4. 🔌 Добавить развязывающие конденсаторы

### **Шаг 3: Разводка сигналов**
1. 📡 Развести I2S сигналы (BCLK, DIN, LRCK, MCLK)
2. 📡 Развести I2C (SDA, SCL) к дисплею
3. 📡 Развести SPI к Flash памяти
4. 📡 Развести GPIO к кнопкам и энкодеру

### **Шаг 4: Разводка аудио**
1. 🎵 Развести аудио выходы от PCM5102A
2. 🎵 Добавить аудио конденсаторы
3. 🎵 Развести аудио разъем
4. 🎵 Проверить аналоговую землю

### **Шаг 5: Разводка USB**
1. 🔌 Развести USB_D+ и USB_D-
2. 🔌 Соблюсти дифференциальную пару
3. 🔌 Добавить фильтрующие конденсаторы

### **Шаг 6: Финальная проверка**
1. 🔍 Запустить DRC проверку
2. 🔍 Проверить все соединения
3. 🔍 Убедиться в отсутствии ошибок
4. 🔍 Проверить 3D вид

## 📊 Спецификация PCB

### **Физические параметры:**
- **Размер:** 90mm x 90mm
- **Толщина:** 1.6mm FR4
- **Слои:** 2 (Top + Bottom)
- **Материал:** FR4, TG 130°C
- **Покрытие:** HASL без свинца
- **Цвет:** Зеленая паяльная маска, белый шелкограф

### **Электрические параметры:**
- **Напряжение:** 5V вход, 3.3V рабочее
- **Ток:** до 2A
- **Импеданс:** 50Ω для USB, 75Ω для аудио
- **Частота:** до 96kHz аудио

### **Компоненты:**
- **SMD компоненты:** 0402, 0603, TSSOP-20
- **Сквозные компоненты:** USB-C, аудио разъем, разъем программатора
- **Монтажные отверстия:** 4x M3

## 🚀 Подготовка к производству

### **Генерация Gerber файлов:**
1. 📁 File → Plot
2. 📁 Выбрать все слои
3. 📁 Включить "Gerber X2 format"
4. 📁 Создать папку "gerber"
5. 📁 Нажать "Plot"

### **Генерация сверловочных файлов:**
1. 📁 File → Plot
2. 📁 Выбрать "Generate Drill Files"
3. 📁 Формат "Excellon"
4. 📁 Нажать "Generate Drill Files"

### **Проверка перед отправкой:**
1. 🔍 Открыть Gerber Viewer
2. 🔍 Проверить все слои
3. 🔍 Убедиться в правильности размеров
4. 🔍 Проверить сверловку

## 📞 Поддержка и отладка

### **Возможные проблемы:**
1. **❌ DRC ошибки:**
   - Проверить зазоры
   - Увеличить ширину дорожек
   - Переместить компоненты

2. **❌ Обрывы соединений:**
   - Проверить net list
   - Убедиться в правильности footprint'ов
   - Проверить назначение пинов

3. **❌ Проблемы с питанием:**
   - Проверить все цепи питания
   - Убедиться в правильной полярности
   - Добавить развязывающие конденсаторы

### **Инструменты для отладки:**
- **KiCad:** 3D Viewer, DRC, ERC
- **Gerber Viewer:** Проверка производственных файлов
- **Осциллограф:** Проверка сигналов
- **Мультиметр:** Проверка питания и соединений
